Collections & Lessons
{% resource "30 Seconds of Code" "https://www.30secondsofcode.org" %} A lot of useful code snippets for Javascript, CSS, and more. {% endresource %} {% resource "Evernote.Design" "https://www.evernote.design" %} Massive collection of design resources and tools. Formatting & Processing
{% resource "Sass" "https://sass-lang.com" %} A preprocessor that extends CSS with variables, mixins, and more. I use this for almost all my CSS work. {% endresource %} {% resource "Prettier" "https://prettier.io" %} An "opinionated code formatter" that supports a variety of different languages. {% endresource %} {% resource "Autoprefixer" "https://autoprefixer.github.io" %} A tool that'll automatically add vendor prefixes into your CSS for wide browser support. {% endresource %}
Frameworks
{% resource "Bootstrap" "https://getbootstrap.com" %} Popular framework primarily for mobile responsive design. I also used parts of Bootstrap (v5.3) to create this site. {% endresource %} {% resource "Tailwind" "https://tailwindcss.com" %} Utility-first; doesn't have pre-defined components like Bootstrap does and instead you build everything with classes. {% endresource %} {% resource "Chota" "https://jenil.github.io/chota/" %} A very tiny, lightweight framework. {% endresource %} {% resource "NES.css" "https://nostalgic-css.github.io/NES.css/" %} A CSS framework in NES style. {% endresource %} {% resource "98.css" "https://jdan.github.io/98.css/" %} A CSS framework in the style of Windows 98. {% endresource %} {% resource "XP.css" "https://botoxparty.github.io/XP.css/" %} A CSS framework in the style of Windows XP. {% endresource %} {% resource "7.css" "https://khang-nd.github.io/7.css/" %} A CSS framework in the style of Windows 7. {% endresource %}
Fonts
{% resource "Google Fonts" "https://fonts.google.com" %} Good old Google Fonts. A collection of free fonts that can be used commercially. {% endresource %} {% resource "Adobe Fonts" "https://fonts.adobe.com/fonts" %} Adobe's free font library. These can also be used commercially. Some may require a Creative Cloud subscription. {% endresource %} {% resource "Font Squirrel" "https://www.fontsquirrel.com" %} More free fonts. Their webfont generator is especially useful for converting fonts to woff/woff2. {% endresource %} {% resource "Fontjoy" "https://fontjoy.com" %} Generate font pairings based on contrast/similarity. Uses Google Fonts. {% endresource %} {% resource "Wakamai Fondue" "https://wakamaifondue.com" %} A site that'll take whatever font you drop in it and give you an analysis. {% endresource %} {% resource "Typography Handbook" "https://typographyhandbook.com" %} Some reading on typography, choosing fonts, using them, etc. {% endresource %}
