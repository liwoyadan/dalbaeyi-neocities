Accessibility
{% resource "W3C Accessibility Principles" "https://www.w3.org/WAI/fundamentals/accessibility-principles/" %} An introduction to web accessibility requirements. Not a complete list, but good to start with. {% endresource %} {% resource "Accessiblity Cheatsheet" "https://moritzgiessmann.de/accessibility-cheatsheet/" %} An easy to read cheatsheet on web accessibility. {% endresource %} {% resource "Accessible Colors" "https://accessible-colors.com" %} A site that will evaluate the contrast ratio of your chosen text and background colour and tells you if it passes AA or AAA guidelines. If it fails, the site will tell you the closest accessible colour combination. {% endresource %} {% resource "Can I Use" "https://caniuse.com" %} You can check browser compatability for various features here. {% endresource %}
Collections & Lessons
{% resource "W3Schools" "https://www.w3schools.com" %} Great site to use as a reference or to learn various coding languages. {% endresource %} {% resource "Web Design Museum" "https://www.webdesignmuseum.org" %} Screnshots of sites from the 90s to 00s. Great for inspiration. {% endresource %} {% resource "Screenlane" "https://screenlane.com" %} Although no longer actively maintained, this is a good library for mobile design inspiration. {% endresource %} {% resource "Evernote.Design" "https://www.evernote.design" %} Massive collection of design resources and tools. Some links are broken or defunct, though. {% endresource %} {% resource "Smashing Magazine - SVG Generators" "https://www.smashingmagazine.com/2021/03/svg-generators/" %} An article with links to dozens of tools for SVG generation and manipulation. {% endresource %} {% resource "Web Dev For Beginners" "https://microsoft.github.io/Web-Dev-For-Beginners/#/" %} Microsoft's free beginner web development curriculum, which features 24 lessons. {% endresource %} {% resource "web.dev" "https://web.dev/learn/" %} Various free courses on topics such as CSS, accessibility, privacy, and more. {% endresource %} {% resource "30 Days of Javascript" "https://github.com/Asabeneh/30-Days-Of-JavaScript" %} A free 30-day curriculum for getting a handle on JavaScript. {% endresource %} {% resource "Flexbox Froggy" "https://flexboxfroggy.com" %} A simple browser game that will teach you the basics of flexbox. {% endresource %} {% resource "CSSBattle" "https://cssbattle.dev" %} Multiplayer game in which you complete CSS challenges to generate a score. A fun way to practice. {% endresource %}
Formatting & Processing
{% resource "Sass" "https://sass-lang.com" %} A preprocessor that extends CSS with variables, mixins, and more. I use this for almost all my CSS work. {% endresource %} {% resource "Prettier" "https://prettier.io" %} An "opinionated code formatter" that supports a variety of different languages. {% endresource %} {% resource "Autoprefixer" "https://autoprefixer.github.io" %} A tool that'll automatically add vendor prefixes into your CSS for wide browser support. {% endresource %} {% resource "Online CSS Minifier" "https://www.toptal.com/developers/cssminifier" %} Page that will minify your CSS for you. Also has API access. {% endresource %}
Frameworks
{% resource "Bootstrap" "https://getbootstrap.com" %} Popular framework primarily for mobile responsive design. I also used parts of Bootstrap (v5.3) to create this site. {% endresource %} {% resource "Tailwind" "https://tailwindcss.com" %} Utility-first; doesn't have pre-defined components like Bootstrap does and instead you build everything with classes. {% endresource %} {% resource "Chota" "https://jenil.github.io/chota/" %} A very tiny, lightweight framework. {% endresource %} {% resource "NES.css" "https://nostalgic-css.github.io/NES.css/" %} A CSS framework in NES style. {% endresource %} {% resource "98.css" "https://jdan.github.io/98.css/" %} A CSS framework in the style of Windows 98. {% endresource %} {% resource "XP.css" "https://botoxparty.github.io/XP.css/" %} A CSS framework in the style of Windows XP. {% endresource %} {% resource "7.css" "https://khang-nd.github.io/7.css/" %} A CSS framework in the style of Windows 7. {% endresource %}
Javascript
{% resource "jQuery" "https://jquery.com" %} Popular JavaScript library, also used by Bootstrap. {% endresource %} {% resource "jQuery UI" "https://jqueryui.com" %} Addendum to the above, a library of neat interactions and effects, such as making divs draggable. {% endresource %} {% resource "anime.js" "https://animejs.com" %} A JavaScript animation library that works with CSS properties, SVGs, DOM attributes, and JS objects. {% endresource %} {% resource "ztext.js" "https://bennettfeely.com/ztext/" %} Small library for creating 3 dimensional text, SVGs, and images. {% endresource %} {% resource "css-doodle" "https://css-doodle.com" %} Web component for creating drawings and patterns with CSS. {% endresource %} {% resource "Masonry" "https://masonry.desandro.com" %} For grid layouts; especially useful for making galleries. {% endresource %} {% resource "Leaflet" "https://leafletjs.com" %} A library for making mobile-friendly interactive maps. {% endresource %} {% resource "PhaserJS" "https://phaser.io" %} An open source framework for making HTML5 browser games. {% endresource %} {% resource "RenJS" "https://renjs.net" %} A visual novel engine using PhaserJS that renders an HTML5 visual novel in-browser out of yaml files. {% endresource %}
Designwork
{% resource "Unsplash" "https://unsplash.com" %} Thousands of free to use photos; I use this site for most of my site backgrounds. {% endresource %} {% resource "Pexels" "https://www.pexels.com" %} Like the above; more free to use stock photos for backgrounds and the like. {% endresource %} {% resource "rawpixel" "https://www.rawpixel.com" %} Has free and public domain images that can be used in projects. Note that rawpixel retains copyright on free images. {% endresource %} {% resource "Haikei" "https://app.haikei.app" %} An app that allows you to generate all sorts of shapes and blobs and save them as either SVG or PNG. {% endresource %} {% resource "Subtle Patterns" "https://www.toptal.com/designers/subtlepatterns/" %} Tiling patterns that can be used as backgrounds. {% endresource %} {% resource "Transparent Textures" "https://www.transparenttextures.com" %} Addendum to the above, dozens of tiling patterns turned transparent. {% endresource %} {% resource "Hero Patterns" "https://heropatterns.com" %} A collection of tiling SVG backgrounds. I use a few backgrounds from here on this site. {% endresource %} {% resource "CSS3 Patterns Gallery" "https://projects.verou.me/css3patterns/" %} A bunch of CSS pattern backgrounds under MIT license. {% endresource %} {% resource "FontAwesome" "https://fontawesome.com" %} Huge icon library with thousands of icons. There are free and paid icon kits. I use it on this site. {% endresource %} {% resource "Icones" "https://icones.js.org" %} Another expansive icon library under MIT license. {% endresource %} {% resource "Textures @ Neocities" "https://textures.neocities.org" %} A few thousand textures that can be used either on 3D models or on sites. {% endresource %} {% resource "99 Gif Shop" "https://99gifshop.neocities.org" %} Massive collection of old web style gifs. {% endresource %}
Colours
{% resource "Colormind" "http://colormind.io/bootstrap/" %} A site that lets you generate palettes and preview them in action. {% endresource %} {% resource "Realtime Colors" "https://realtimecolors.com/" %} Another site where you can preview your chosen colours live on a page. {% endresource %} {% resource "ColorSpace" "https://mycolor.space" %} Enter a single colour and the site will generate a variety of palettes for you. {% endresource %} {% resource "Super Color Palette" "https://supercolorpalette.com/" %} Colour palette generator where you can individually adjust hue, saturation, and lightness shifts. {% endresource %} {% resource "Coolors" "https://coolors.co" %} Easy colour palette generator, as well as having thousands of premade palettes in their gallery. {% endresource %}
Fonts
{% resource "Google Fonts" "https://fonts.google.com" %} Good old Google Fonts. A collection of free fonts that can be used commercially. {% endresource %} {% resource "Adobe Fonts" "https://fonts.adobe.com/fonts" %} Adobe's free font library. These can also be used commercially. Some may require a Creative Cloud subscription. {% endresource %} {% resource "Font Squirrel" "https://www.fontsquirrel.com" %} More free fonts. Their webfont generator is especially useful for converting fonts to woff/woff2. {% endresource %} {% resource "Fontjoy" "https://fontjoy.com" %} Generate font pairings based on contrast/similarity. Uses Google Fonts. {% endresource %} {% resource "Fontpair" "https://www.fontpair.co" %} Collection of font pairings from across the web. {% endresource %} {% resource "Wakamai Fondue" "https://wakamaifondue.com" %} A site that'll take whatever font you drop in it and give you an analysis. {% endresource %} {% resource "Typography Handbook" "https://typographyhandbook.com" %} Some reading on typography, choosing fonts, using them, etc. {% endresource %}

Tools & Generators
{% resource "Animista" "https://animista.net" %} CSS animations library with a wide array of effects that can be edited on-site. {% endresource %} {% resource "Clippy" "https://bennettfeely.com/clippy/" %} Classic clip path generator with a handful of presets you can load and edit. {% endresource %} {% resource "UpLabs Clip Path Generator" "https://www.uplabs.com/posts/clip-path-generator" %} Another clip path generator. {% endresource %} {% resource "Fancy-Border-Radius" "https://9elements.github.io/fancy-border-radius/" %} Border-radius generator for fun shapes and curves, using percentages. {% endresource %} {% resource "CSS Gradient" "https://cssgradient.io" %} Easy tool for making gradients that also has a maximum browser compatibility setting for its output. {% endresource %} {% resource "Shadow Palette Generator" "https://www.joshwcomeau.com/shadow-palette/" %} A generator for box shadows that previews the shadows at three different elevations. {% endresource %} {% resource "Patternify" "http://www.patternify.com" %} Pixel-by-pixel tiling background generator; note that the canvas is rather small. {% endresource %} {% resource "Neumorphism Generator" "https://neumorphism.io/" %} Generator for neumorphic design. {% endresource %} {% resource "imagemaps" "https://www.imagemaps.net" %} A tool to create annotated interactive images. {% endresource %} {% resource "Favicon Generator" "https://realfavicongenerator.net" %} Generates favicon files out of a chosen image along with other icons such as iOS/Android. {% endresource %} {% resource "CSS Portal" "https://www.cssportal.com" %} A huge collection of generators, recourses, and tools alongside a reference to CSS functions/properties/etc. {% endresource %} {% resource "CSS Generators" "https://css-generators.com" %} A collection of tools mostly for generating custom borders and shapes through masks and clip paths. {% endresource %} {% resource "Omatsuri" "https://omatsuri.app" %} An app that has a variety of tools from triangles to gradients to even fake data generation. {% endresource %} {% resource "10015" "https://10015.io/tools/css-background-pattern-generator" %} Another collection of various tools from lorem ipsum to CSS to minifiers. Link goes to their CSS background pattern generator, which I particularly like. {% endresource %} {% resource "DOODAD" "https://doodad.dev" %} Has a background pattern and gradient generator, but more interestingly, a converter for images that uses dithering to reduce their file size. {% endresource %} {% resource "CSS Background Patterns" "https://www.magicpattern.design/tools/css-backgrounds" %} Pure CSS background patterns. Allows you to adjust colour, opacity, and spacing of the patterns. {% endresource %}
